# 🍻 のみゲー — 飲み会ゲームアプリ

負けた人が飲む、スマホ1台で盛り上がる飲み会ゲーム集。
**インストール不要・オフライン動作・ビルド不要**（単一HTML）。

🔗 **公開URL: https://nomigee.vercel.app**
（スマホで開いて「ホーム画面に追加」→ アプリのように全画面起動・オフライン対応）

> デプロイは GitHub（`main` ブランチ）→ Vercel 連携の自動デプロイ。`git push` するたびに本番へ反映されます。

## 収録ゲーム（5種）

| ゲーム | 内容 |
|---|---|
| 🎯 **ルーレット** | 登録メンバーで運命の輪を回す。止まった人が飲む。 |
| 💣 **時限爆弾** | お題に答えて隣へ手渡し。ランダムな時間(10〜45秒)で爆発、持ってた人が飲む。爆発音＋バイブ付き。 |
| 🍺 **ロシアンルーレット** | 4/6/9個のコップから順番に選択。アタリを引いた人が飲む。 |
| ⏱️ **ぴったりSTOP** | 数字が見えない状態で10.00秒ぴったりを狙う。一番ズレた人が飲む。 |
| 🎴 **お題ガチャ** | 真実・命令・一気・全員のお題をランダムに引く。約60種。 |

メンバー登録はルーレットと共通。ブラウザに自動保存されます。

## 使い方

### いますぐ試す（PC）
このフォルダを簡易サーバで開くだけ:
```
python -m http.server 4399 --directory .
```
→ ブラウザで http://localhost:4399

### スマホで使う（おすすめ）
1. このフォルダを **Vercel / Netlify / GitHub Pages** にそのままドラッグ＆ドロップでデプロイ
2. スマホでURLを開く
3. ブラウザの共有メニューから **「ホーム画面に追加」**
   → アプリのように全画面起動・**電波が無くても動く**（PWA対応）

> 会場のWi-Fiが弱くても一度開けばオフラインで動くので、飲み会本番でも安心。

## 構成

```

```
